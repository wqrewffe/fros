How to Use NAFAI Effectively

  • Global Search: Type normally, but it might take a little time. Do not use a dot (.).
  • Generative AI: Start your query with a dot (.) to get smart answers.
  • Latest News: Use the keyword news.
  • Math: Use keywords like solve, calculate, or compute.
  • Weather: Use the keyword weather.
  • Dictionary: Use define or meaning of. (⚠️ This feature is still under development.)
